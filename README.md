## Instalação

Clone o repositório:

```git
git clone https://github.com/gcairesdev/dev.finances
```

Entre na pasta do projeto:

```sh
cd dev.finances
```

Instale as dependências:

```sh
npm i
```

Execute o projeto em modo desenvolvimento:

```sh
npm run dev
```
